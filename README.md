# 👋 Hello World

My name is Artur Flis and I currently live in Poznań, Greater Poland.

I’ve been working as a frontend developer since 2020. My main focus is UI/UX - I care a lot about clean layout, good visual balance, and making interfaces feel polished. Sometimes I also like to go off the rails and build purely aesthetic websites just for fun.

On the frontend I mainly work with CSS and JavaScript, and I’m currently planning to learn Vue to expand my stack. Outside of frontend work, my favorite programming language is Go because it’s fast, easy to understand, and compiles into a single binary, which makes it great for building practical tools and backend services.

I’m currently studying IT in a technikum and I regularly take part in different IT olympiads and competitions.

In my free time I’m into self-hosting and home infrastructure - I run my own server at home. I also enjoy flying FPV drones (still learning), 3D printing, and I’m in the process of creating my own podcast.
